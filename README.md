Практическая методичка по XPath/XPath 2.0, подготовленная для задач
промышленного парсинга HTML и XML (СМИ, сайты, структурированные данные).

Материал основан на реальной практике:

условия (if-then-else)
итерации (for, every/satisfies)
регулярные выражения
очистка и нормализация данных
устойчивые селекторы под меняющуюся верстку
Документ предназначен для демонстрации уровня владения XPath 2.0.
Все примеры обезличены и не содержат данных конкретных проектов.
